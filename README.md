#  CRUD Étudiants — Symfony

Application web de gestion des dossiers étudiants développée avec le framework **Symfony**.  
Elle permet de créer, lire, modifier et supprimer des fiches étudiants.

---

## Fonctionnalités

- Lister tous les étudiants dans un tableau
- Ajouter un nouvel étudiant
- Consulter la fiche détaillée d'un étudiant
- Modifier les informations d'un étudiant
- Supprimer un étudiant avec page de confirmation
- Validation des formulaires côté serveur
- Messages flash de succès / erreur
- Protection CSRF sur la suppression

---

## Technologies utilisées

- **PHP 8.1+**
- **Symfony 6/7**
- **Doctrine ORM**
- **Twig** (moteur de templates)
- **MySQL** (base de données)
- **CSS personnalisé** (sans framework CSS)
- **Google Fonts** — DM Sans + DM Serif Display
